Backend changes
File	Change
backend/app/schemas.py	Added created_by_username to EngagementSummary, uploaded_by_username to AssetSummary. Added EngagementUpdate and AssetUpdate schemas (all fields optional — partial-update semantics).
backend/app/auth.py	New helpers can_edit_engagement(user, eng) and can_edit_asset(user, asset)
backend/app/search.py	_engagement_summary and _asset_summary now include the username fields
backend/app/routers/upload.py	Four new endpoints: PATCH /engagements/{id}, DELETE /engagements/{id}, PATCH /assets/{id}, DELETE /assets/{id}. Engagement delete cascades through SQLAlchemy (assets → chunks/tags, lessons), explicitly wipes the junction tables (platforms, tech stack, cloud, llms, contributors), and best-effort cleans up local files + ChromaDB vectors. Asset delete cleans up its file + vectors. All endpoints log to audit_log.
